About This Project

The Medicine Recommendation System was developed as a way to apply classroom learning to a real-world problem. It predicts possible diseases based on user symptoms using a trained ML model, while also offering useful medical recommendations.

Technologies Used in This Project

This system was built using Python, Flask, scikit-learn, Bootstrap, and basic HTML/CSS. The goal was to make healthcare intelligence simple, helpful, and accessible to users with minimal technical knowledge.
